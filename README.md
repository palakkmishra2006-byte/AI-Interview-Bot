# AI Interview Bot 🤖

An AI-powered interview preparation system built using Python and Google's Gemini API.

## Features

- Generate dynamic interview questions
- Evaluate candidate answers using AI
- Provide scores and detailed feedback
- Identify strengths and areas for improvement
- Simulate a real interview experience

## Technologies Used

- Python
- Gemini API
- VS Code
- Git & GitHub

## Project Versions

### Version 1
- Basic interview questions

### Version 2
- Stores and displays answers

### Version 3
- AI-generated questions and feedback using Gemini

## Future Enhancements

- Flask Web Application
- User Authentication
- Interview History
- Performance Dashboard
- Online Deployment
